# 🏥 Raio-X SUS x IBGE: O Impacto Socioeducacional na Saúde

Este projeto de Business Intelligence analisa os dados de atendimentos do SUS no estado do Ceará, cruzando-os com indicadores socioeducacionais (População, IDEB e Escolarização) do IBGE.

### 🛠️ Tecnologias Utilizadas
* **Python** (Pandas, Numpy) para ETL e Engenharia de Dados.
* **Plotly** para visualização interativa e mapas.
* **Streamlit** para criação do dashboard web analítico.

### 📊 Principais Descobertas
* Comprovação de "Cidades Polo" que operam acima da capacidade populacional.
* Correlação estatística de Pearson negativa entre a melhoria do IDEB e a redução proporcional da pressão em urgências do SUS.

### 🚀 Como executar o projeto localmente
1. Clone o repositório: `git clone https://github.com/SEU-USUARIO/analise-sus-ibge-ceara.git`
2. Instale as dependências: `pip install -r requirements.txt`
3. Execute a aplicação: `streamlit run app.py`
